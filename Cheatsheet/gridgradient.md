# Crazy Things You Can Do with CSS Gradients
Next-level css gradients:

you may think we can just make this with gradient for following div:

```html
<div class="test"></div>
```


this
<table width="100%">
  <tr>
<td>
<div style="width:100%; height:200px; background:linear-gradient(black, red); border-radius:20px;"></div>
</td>
  </tr>
</table>


```css
.test {
  height: 200px;
  background-image: linear-gradient(black, red);
}
```
